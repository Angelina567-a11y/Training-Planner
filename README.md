# Training Planner

Автор: Элиза Валеева

## Описание
Это GUI-приложение для планирования тренировок. Вы можете добавлять тренировки, фильтровать их по дате и типу, сохранять и загружать из JSON.

## Как запустить
1. Установите Python 3
2. Запустите файл `code.py`
3. Используйте интерфейс для добавления, фильтрации и сохранения тренировок.

## Пример использования
- Введите дату в формате ГГГГ-ММ-ДД, тип тренировки и длительность.
- Нажмите «Добавить тренировку».
- Для фильтрации введите критерии и нажмите «Фильтровать».
- Для сохранения — «Сохранить в JSON».
- Для загрузки — «Загрузить из JSON».
